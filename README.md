# CoreDumps

Simulation d’une compétition entre 16 équipes nationales grâce
à la simulation des différents matchs avec des résultats basés sur les caractéristiques de chaque équipe.

## Conçu avec

[Eclipse](https://www.eclipse.org/) - IDE
